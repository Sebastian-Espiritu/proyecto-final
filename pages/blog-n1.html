    • UI Developer / Diseñador UI
    • Web Designer / Maquetador
    • Programador Front-end
  3. Resumen

Para finalizar, estos roles van a depender de una organización, del presupuesto, y de otras cosas, ya que pueden llegar a superponerse unos a otros, pero no es recomendable que una sola persona se encargue de todo el trabajo, no en estos tiempos y menos si se quiere tener un sitio web bien estructurado y optimizado. Puede llegar a suceder que, un Back-end Developer sea a la vez el Web Designer del sitio, pero que un UI Developer sea el Programador Front-end al mismo tiempo no es para nada viable, no a nuestro parecer.
